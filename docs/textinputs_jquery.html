Overview

Rangy Text Inputs is a small jQuery plug-in for manipulating the selection and value of HTML textareas and text inputs. Unlike similar plug-ins, Rangy Text Inputs works correctly in all situations in Internet Explorer 6-8, including in textareas with leading or trailing empty lines.

Rangy is a modular, cross-browser JavaScript range and selection library. It provides a simple API for performing common range and selection tasks in all major browsers, abstracting away the wildly different implementations of this functionality between Internet Explorer and DOM-compliant browsers.

API

The following are added as methods of every jQuery object that wraps a textarea or text input:

  • Object getSelection()

    Retrieves an object representing the selection within the text input, containing the following properties:

    • start
      The position in the text input's value where the selection starts.
    • end
      The position in the text input's value where the selection ends.
    • length
      The length of the selection. If the selection is an insertion point (caret), this property's value is 0.
    • text
      The portion of the text input's text that is selected.
  • Element setSelection(Number start, Number end)
    Sets the selected portion of the text input and returns the text input. A negative number for start or end is treated as an offset relative to the end of the text input (e.g. -2 is treated as two characters from the end).
    • start
      The start position within the text input's text of the new selection.
    • end
      The end position withing the text input's text of the new selection.
  • Element collapseSelection(Boolean toStart)
    Collapses the selection within the text input to a caret at the either the start or the end of the selection (depending on toStart) and returns the text input.
  • Element deleteSelectedText()
    Deletes the selected text from the text input, collapses the selection to a caret at the point from which the text was deleted and returns the text input.
  • String extractSelectedText()
    Deletes the selected text from the text input, collapses the selection to a caret at the point from which the text was deleted and returns the text that was deleted.
  • Element replaceSelectedText(String text)
    Replaces the selected text in the text input with the text provided, collapses the selection to a caret at the end of the replacement text and returns the text input.
  • Element surroundSelectedText(String before, String after)
    Surrounds the selected text in the text input with the text provided, moves the selection to include the surrounded text and returns the text input.
  • Element insertText(String text, Number index, Boolean moveSelection)
    Inserts text at position index in the text input's value, optionally moves the caret to the end of the inserted text (depending on moveSelection) and returns the text input. A negative number for index is treated as an offset relative to the end of the text input (e.g. -2 is treated as two characters from the end).
  • Element deleteText(Number start, Number end, Boolean moveSelection)
    Deletes text at between positions start and end in the text input's value, optionally moves the caret to the point at which the text was deleted (depending on moveSelection) and returns the text input. A negative number for start or end is treated as an offset relative to the end of the text input (e.g. -2 is treated as two characters from the end).
