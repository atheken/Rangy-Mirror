Introduction

Rangy is a modular, cross-browser JavaScript range and selection library. It provides a simple API for performing common range and selection tasks in all major browsers, abstracting away the wildly different implementations of this functionality between Internet Explorer and DOM-compliant browsers.

Rangy's main features are:

  • A full* implementation of the DOM Level 2 Range specification
    • in all supported browsers, including IE 6 and above
  • A full* implementation of the HTML 5 Selection interface in all supported browsers, including IE 6 and above
  • Custom extensions to Range and Selection API
  • API providing easy traversal of nodes contained within a Range
  • Range contents mutation API, including built-in API for applying and removing CSS classes to content within a Range
  • Standalone text inputs API
