Range public API

A Range in Rangy implements every property and method of DOM Level 2 Range. It also contains extra methods.

Read-only properties

Any property marked as read-only below is not in fact read-only in that code may modify it without error. However, Rangy expects such a property not to be modified by client code; if the property is modified then Rangy's behaviour is undefined and probably undesirable.

The reason for this limitation is that it isn't possible to create read-only object properties in all of Rangy's target browsers. Rangy is as consistent as possible in its target browsers which unfortunately means it has to work to the lowest common denominator.

Properties

  • startContainer

    Read-only. DOM 2 MDC

    Returns the Node within which the Range starts. See the note above about read-only properties.
